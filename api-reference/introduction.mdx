## What the API is for

The Pierview API (coming soon) will enable:

- **Automation** — Trigger scans, fetch results, and sync data into your tools
- **Exports** — Pull visibility, sentiment, and citation data into spreadsheets or BI
- **Integrations** — Connect with CRMs, content platforms, and reporting dashboards
